Functional Requirements 
1. Users must be able to login with Facebook Account.
2. Logged in users must be able to select their joined groups.
3. Logged in users must be able to see posts from groups.
4. Logged in users must be able to sort posts by date or any other metadata available from Facebook (location, etc) 
5. Users must have a way of rating seller and post- this would be stored on a database hosted on my server.
